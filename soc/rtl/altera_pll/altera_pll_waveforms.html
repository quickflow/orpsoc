Sample behavioral waveforms for design file altera_pll.v

The following waveforms show the behavior of altpll megafunction for the chosen set of parameters in design altera_pll.v. The design altera_pll.v has Cyclone III AUTO pll configured in NORMAL mode The primary clock input to the PLL is INCLK0, with clock period 20000 ps. Output port LOCKED will go high when the PLL locks to the input clock.
